# cs-form-mtn-011-vs2022

所属マスタを作成して、そこから一覧を選択可能にすめコンボボックスの実装

```sql
CREATE TABLE `所属マスタ` (
  `所属コード` char(4) NOT NULL,
  `所属名称` varchar(50) DEFAULT NULL,
  PRIMARY KEY (`所属コード`)
)
```
